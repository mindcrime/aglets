Aglets SDK Install

Prerequisites

To Install:

  • The install requires a working installation of the  Jakarta Project's Ant .

To Build

  • Jakarta's Log4j
  • Either  Jaxp  or Xerces .
  • Jar files for Log4j and the XML parser should be placed in the src/lib directory or in the system classpath.

Installation Instructions

  1. Change to the bin directory.
  2. Run the command:  ant
  3. Follow the instructions given for installing a keystore and policy file.

To run the server with the default configuration file:

  • NT:    agletsd -f ..\cnf\aglets.properties
  • Unix: ./agletsd -f ../cnf/aglets.properties
